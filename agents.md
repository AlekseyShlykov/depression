# Agents — The Earthling

## Что это

Прототип визуальной новеллы для **GitHub Pages**: чистый **HTML/CSS/JS**, без сборщика. Два инопланетянина, диалог из **Twine → JSON** (`dialogues/ru.json`, `dialogues/en.json`), UI-строки в **`locales/*.json`**. Печать текста посимвольно, тап по экрану допечатывает строку, в конце истории без выборов — форма email (**Formspree**, только поле `email`).

## Сценарий из Twine

Подробно: **`TWINE_DIALOGUE_RULES.md`**. Кратко:

- **Имена в тексте реплик не писать** — в JSON только префиксы **`Alien1:`** / **`Alien2:`** и фраза после двоеточия; кто говорит, видно по подсветке портрета.
- Строка **`//…`** (целая строка, непусто после `//`) — **служебная команда**: выполняется, **не показывается** в диалоге (`runStoryCommand` в `js/main.js`).
- Формат экспорта **«Twine to JSON»**: `cleanText`, `links` с `linkText` / `passageName`; **первый пассаж** в массиве — старт.

## Разработка дальше

- Менять парсинг истории — **`js/twison.js`**; команды `//` — **`js/main.js`**.
- Не раздувать дифф без нужды; стиль и структура как в существующих файлах.
- Новые UI-строки — в **оба** `locales/en.json` и `locales/ru.json`.
- Одновременная речь двух героев — несколько пассажей подряд, не одна «каша» в одном пассаже (см. README / `TWINE_DIALOGUE_RULES.md`).
